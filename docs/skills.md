# Skills Index

## Data API

Skills for running reports and retrieving analytics data from the GA4 Data API.

| Skill | Description |
|-------|-------------|
| [ga4-reports](../skills/ga4-reports/SKILL.md) | Run GA4 Data API reports with dimensions, metrics, filters, and pagination. Always outputs JSON aligned with the RunReportResponse structure. |
| [ga4-realtime](../skills/ga4-realtime/SKILL.md) | Run GA4 realtime reports to get live user and event data. Always outputs JSON. |
| [ga4-metadata](../skills/ga4-metadata/SKILL.md) | List available dimensions and metrics for a property, and check which combinations are compatible. |

## Admin API

Skills for managing GA4 account resources via the Analytics Admin API.

| Skill | Description |
|-------|-------------|
| [ga4-accounts](../skills/ga4-accounts/SKILL.md) | List and inspect GA4 accounts accessible to the authenticated credential. |
| [ga4-properties](../skills/ga4-properties/SKILL.md) | List and inspect GA4 properties within an account. |
| [ga4-datastreams](../skills/ga4-datastreams/SKILL.md) | List and inspect data streams for a GA4 property. |
| [ga4-key-events](../skills/ga4-key-events/SKILL.md) | List key events (formerly conversions) configured on a GA4 property. |
| [ga4-custom-dimensions](../skills/ga4-custom-dimensions/SKILL.md) | List custom dimensions registered on a GA4 property. |
| [ga4-custom-metrics](../skills/ga4-custom-metrics/SKILL.md) | List custom metrics registered on a GA4 property. |
| [ga4-audiences](../skills/ga4-audiences/SKILL.md) | List and inspect audiences defined on a GA4 property. |
| [ga4-events](../skills/ga4-events/SKILL.md) | List event create rules for a GA4 data stream. |
